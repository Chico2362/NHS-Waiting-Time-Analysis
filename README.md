# NHS Waiting Time & Service Performance Analysis

## Project Overview
This project analyses NHS England waiting time datasets to evaluate compliance with the 18-week Referral to Treatment (RTT) target and identify regional performance differences.

## Objectives
- Assess RTT performance across NHS regions
- Identify trends in waiting times
- Analyse emergency admissions patterns
- Build an interactive dashboard for monitoring performance

## Dataset
Source: NHS England Waiting Time Data  
https://www.england.nhs.uk/statistics/statistical-work-areas/rtt-waiting-times/

Records analysed: 100,000+

## Tools Used
- SQL
- Python (Pandas, NumPy)
- Power BI

## Analysis Performed
- Data cleaning and transformation
- Regional RTT comparison
- Time series trend analysis
- Statistical testing for regional variation

## Dashboard KPIs
- RTT compliance rate
- Emergency admissions
- Monthly waiting time trends
- Regional performance comparison

## Key Findings
- Significant regional variation in RTT compliance
- Seasonal spikes in emergency admissions
